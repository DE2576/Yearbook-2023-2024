Ordered and Unordered Lists

On webpages, we can make lists that are numbered or use a dot (not numbered).
If we want them numbered, we use the ordered list (ol) tag.
If we want no numbers, we use the unordered (ul) tag.

Ordered List Example

  1. Apples
  2. Bananas
  3. Peaches
  4. Tomatos
  5. Zuchinis

Unordered List Example

  • SiFu
  • Hollow Knight
  • Metroid Dread
